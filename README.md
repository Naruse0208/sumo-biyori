# 相撲日和 / Sumo Biyori

大相撲の取組速報、力士レーティング、勝率予測、AIによる見どころ解説を日本語・英語で楽しめる非公式ファンサイトです。

## 主な機能

- 当日の全六段の取組進行、結果、次の一番
- 1958年以降の取組から再計算した Elo・Glicko-2・相撲偏差値
- 力士プロフィール、レート推移、対戦比較、歴代横綱比較
- 事前勝率と予測モデルの公開検証
- DBの事実データだけを渡して生成する、日英3項目のAI見どころ解説
- 日本語／英語表示、PC・スマートフォン対応

## AI見どころ生成

生成先は環境変数で切り替えられます。現在の本番設定は OpenAI の `gpt-5.6-luna` です。生成済みの文章はモデル変更時にも保持し、翌日の新しい取組から新設定を使います。

```env
AI_PROVIDER=openai
OPENAI_MODEL=gpt-5.6-luna
OPENAI_API_KEY=your_api_key
```

Geminiを使う場合は `AI_PROVIDER=gemini` と `GEMINI_API_KEY` を設定してください。APIキーはブラウザへ送らず、公開リポジトリにも含めません。

## ローカル起動

Node.js 22.13以上が必要です。

```bash
npm install
npm run dev
```

検証用コマンド:

```bash
npm run build
node --test tests/rendered-html.test.mjs
```

## データと免責

取組・番付・力士プロフィールの出典は日本相撲協会公式サイトです。レーティング、勝率、歴代指数、AI解説は相撲日和独自の推定・編集情報であり、日本相撲協会の公式見解ではありません。

本サイトは非公式ファンプロジェクトです。予測値は観戦を楽しむための参考情報であり、賭博その他の金銭判断を目的としません。
